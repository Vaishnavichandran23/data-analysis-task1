# 📊 Data Analytics Internship - Task 1

## 🔹 Project Title

Data Immersion & Wrangling

## 👩‍💻 About Me

Hi, I am Vaishnavi, currently working on a Data Analytics Internship task.
This project focuses on understanding, cleaning, and preparing raw data for analysis.

## 📁 Dataset

* File: `Customer_Data.csv`
* Description: Contains customer financial and transaction details such as balance, purchases, credit limit, payments, etc.

## 🎯 Objective

To perform:

* Data understanding
* Data cleaning
* Data transformation
* Preparing dataset for analysis

## 🔍 Steps Performed

### 1. Data Understanding

* Loaded dataset using Python (Pandas)
* Explored columns and data types

### 2. Data Quality Assessment

* Identified missing values
* Checked duplicate records
* Detected outliers

### 3. Data Cleaning

* Filled missing values using mean
* Removed duplicate rows
* Handled outliers

### 4. Data Transformation

* Created new features:

  * `avg_purchase`
  * `total_transactions`
* Categorized balance into groups
* Normalized numerical data

## 🛠️ Tools & Technologies

* Python

* VS Code
